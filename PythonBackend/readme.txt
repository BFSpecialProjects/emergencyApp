NestWatch Proof-Of-Concept Backend using Python and Twilio

testing environment:
elementary linux (ubuntu)
anaconda python 2.7
ngrok 2.2.8

Twilio dependencies
* pip (included in anaconda, or from apt)
* twilio (pip install twilio)
* flask (pip install flask)


Setting up the server:
* Make sure environment is set up properly (Flask, Twilio dependencies)
* run twilioCombined.py
* set up ngrok
* be sure to change webhook in Twilio
